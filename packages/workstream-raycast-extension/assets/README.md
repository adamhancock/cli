# Assets

Please add a `command-icon.png` file to this directory.

## Icon Requirements
- Format: PNG
- Size: 512x512 pixels recommended
- Transparent background preferred
- Should represent VS Code window switching

You can create this icon using any design tool or find a suitable icon from:
- SF Symbols (macOS built-in)
- https://www.flaticon.com
- https://icons8.com
